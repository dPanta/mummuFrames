# mummuFrames
Unitframes addon for World of Warcraft.
Minimal, core functionalities, basic customization. Aimed at visual clarity and speed (hopefully)

## Font Catalog
The addon uses `Core/FontCatalog.lua` for the font dropdown list.

After adding/removing `.ttf`/`.otf` files in `Media/`, run:

```bash
./scripts/generate_font_catalog.sh
```
